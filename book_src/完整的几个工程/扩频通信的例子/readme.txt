里面包含的内容包括：
1、完整的物理层通信设计支撑库，包括designware/Cordic库/RS库/FFT库等.
2、完整的Matlab与定点C联合仿真交互引擎，读者可以利用HLS或Matlab进行定点化快速仿真。
